# Age and Emotion Detection Through Voice
Age and Emotion Detection Through Voice is a machine learning-based application that analyzes voice recordings to identify gender, estimate age, detect senior citizens, and recognize emotions. The system first classifies the speaker's gender and rejects female voice inputs by displaying "Upload Male Voice." For male speakers, it predicts age using audio features extracted from voice recordings. If the predicted age is above 60 years, the system identifies the speaker as a senior citizen and performs emotion detection. The project uses AudioMNIST for age and gender prediction, RAVDESS for emotion recognition, Librosa for audio feature extraction, Random Forest machine learning models for classification and regression, and Streamlit for the graphical user interface.
